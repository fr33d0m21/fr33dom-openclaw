OpenClaw VM Admin Shell

A lightweight operator UI for an Ubuntu OpenClaw VM. It links the real OpenClaw Gateway UI with a browser shell so you can onboard, debug, and snapshot a repeatable image.

The OpenClaw Control UI should appear on port 18789 once the gateway is running. The browser terminal is provided separately via `ttyd`.

Bootstrap

    Notes

    • Run `fr33d0m-openclaw onboard --install-daemon` first.
    • The OpenClaw Control UI is served by the Gateway itself.
    • This shell is intended as a branded launcher and status page, not a replacement for the real OpenClaw UI yet.
    • Once validated, snapshot the Ubuntu Desktop VM on DigitalOcean.
